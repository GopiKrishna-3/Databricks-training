# WEEK 3 – DAY 6  
# DENSE_RANK() FUNCTIONS IN MYSQL

This project explains DENSE_RANK() functions using employee and order datasets.

The queries demonstrate:

- Joining date ranking
- Order amount ranking
- City-wise dense ranking

---

# Topics Covered

## 1. Dense Rank Based on Joining Date

Assign dense ranks using joining dates.

## 2. Dense Rank Based on Order Amount

Rank orders based on highest order amount.

## 3. City-wise Dense Ranking

Assign dense ranks within cities using order amount.

---

# SQL Concepts Used

```sql
DENSE_RANK()
OVER()
PARTITION BY
ORDER BY
```

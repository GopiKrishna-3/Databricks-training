# Week 1 - Day 3

## Objective

The objective of this session was to understand and practice SQL Joins using multiple related tables. The session focused on retrieving meaningful data from connected tables using different join operations such as LEFT JOIN, RIGHT JOIN, FULL OUTER JOIN, and CROSS JOIN. The exercises helped strengthen relational database concepts and improve query writing skills for handling real-world data relationships.
